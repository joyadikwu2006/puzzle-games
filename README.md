# 🕹️ Real Bug // Fake Bug

A fast-paced web trivia game about coding history, tech myths, and software lore. Players have 8 seconds to guess if a claim is REAL or FAKE!

## 🎮 Features
* 🧩 **3 Levels:** Easy, Medium, and Hard stages.
* ⏱️ **Countdown Timer:** You only have 8 seconds per question.
* 🔀 **Randomized Questions:** Questions mix up every time you retry.
* 🎨 **Clean Design:** Dark mode theme with fun animations.

## 🛠️ Built With
* HTML5
* CSS3
* Vanilla JavaScript (No frameworks needed!)

## 🚀 How to Play
1. **Clone the project:**
   ```bash
   git clone https://github.com
   ```
2. **Open the game folder:**
   ```bash
   cd real-bug-fake-bug
   ```
3. **Launch the game:**
   Double-click the `index.html` file in your file folder, or open it from your terminal:
   * **Mac:** `open index.html`
   * **Windows:** `start index.html`
   * **Linux:** `xdg-open index.html`
